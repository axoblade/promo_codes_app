## End-point: Create Promo Code
### Description: 
Method: POST
>```
>{{URL}}/api/promo_codes
>```
### Body (**raw**)

```json
{
    "code_name" : "V2BQXT",
    "valid_for" : 5,
    "amount" : 6500,
    "address" : "Mandela National Stadium",
    "longitude" : 32.66,
    "latitude" : 0.35,
    "radius" : 700
}
```
